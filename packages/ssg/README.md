# MDXX SSG

## Features

- Build mdxx on next ssg
- Support AMP

## Create your own blog

```bash
# install node and npm
npx degit mizchi/mdxx/packages/ssg blog
cd blog
npm install
npm run create-item hello-mdxx
# edit docs/hello-mdxx.mdx

# Preview
npm run dev

# Deploy
npm run build
npm run deploy
```
